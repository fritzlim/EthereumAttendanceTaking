Ethereum Developer Course

This project is for completing the 2018 ConsenSys Developer Program

Learn about blockchain technology and develop dApps on the Ethereum platform. See https://www.ethereum.org/ for more.

Intro to Blockchain

0 ETH

  • Learn about the fundamentals of blockchain technology and what its use cases are.
  •  
  • This course is suitable for beginners without any technical knowledge.

Solidity 101

2 ETH

  • Introducing the Solidity programming language for smart contracts.
  •  
  • This course is suitable for software developers who have no prior knowledge of Solidity.

Solidity 102

2 ETH

  • An intermediate course on programming smart contracts using the Solidity language.
  •  
  • To take this course, you need to complete and pass Solidity 101.
